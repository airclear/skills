## 状态说明

- **草稿**: 正在撰写中
- **评审中**: 提交评审，等待反馈
- **已批准**: 通过评审，可进入开发
- **开发中**: 对应 Track 已创建，研发进行中
- **已发布**: 功能已上线
- **已归档**: 功能废弃或重新规划

## 命名规范

- **格式**: `P[三位编号]_[特性简称]`
- **示例**: `P001_user_auth`, `P002_data_export`
