## 状态说明

- **待分析**: 原始需求，尚未进行深入分析
- **分析中**: 正在进行需求调研和方案设计
- **已转化为PRD**: 已完成 PRD 文档，进入 `02_prd/`
- **已拒绝**: 需求被否决，保留记录

## 录入规范

1. **目录命名**: `[三位编号]_[来源类型]_[简短描述]`
2. **必含文件**: `requirement.md`（原始需求描述）
3. **资源目录**: `_resources/` 存放图片等附件
4. **更新索引**: 录入后必须更新本文件
